# @nomercy-entertainment/nomercy-player-kit

> Shared spine for NoMercy player libraries.

Used by:
- [`@nomercy-entertainment/nomercy-music-player`](https://github.com/NoMercy-Entertainment/nomercy-music-player)
- [`@nomercy-entertainment/nomercy-video-player`](https://github.com/NoMercy-Entertainment/nomercy-video-player)

Provides the primitives both players use: plugin runtime, event system, lifecycle helpers, DOM helpers, storage, logger, stream protocol abstraction, and time-indexed cue lists. Domain-specific code (audio backends, crossfade, video element handling, subtitles) lives in each player.

## Status

**0.0.0 — pre-alpha.** API surface is stubbed; no functionality yet. Tracking the design at [`feature/api-unification-consolidation`](https://github.com/NoMercy-Entertainment) across all three repos.
